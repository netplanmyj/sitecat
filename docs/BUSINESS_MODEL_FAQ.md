# Business Model & Pricing FAQ

This document addresses common questions about SiteCat's business model and pricing structure, particularly for App Store review purposes.

## General Business Model

### Current Status (Version 1.0.1)
- **All features are FREE**
- No subscriptions
- No paid content
- No external payment systems
- No fees to create or use an account

## App Store Review Questions & Answers

### 1. Who are the users that will use the paid subscriptions and features in the app?

**Current Answer (v1.0.1):**
There are NO paid subscriptions or features in the current version. All features are completely free for all users.

**Future Plans:**
When premium features are introduced, they will be available to users who want:
- Automatic monitoring (scheduled checks)
- Unlimited site monitoring
- Push notifications
- Advanced analytics

Premium features will be offered through Apple's In-App Purchase system only.

### 2. Where can users purchase the subscriptions and features that can be accessed in the app?

**Current Answer (v1.0.1):**
There are NO subscriptions or paid features to purchase at this time.

**Future Plans:**
All future premium features will be available exclusively through Apple's In-App Purchase system. No external payment methods will be used.

### 3. What specific types of previously purchased subscriptions and features can a user access in the app?

**Current Answer (v1.0.1):**
None. There are no subscriptions or paid features in the current version.

**Future Plans:**
Users who purchase premium features through In-App Purchase will have access to:
- Automatic monitoring schedules
- Unlimited site monitoring
- Push notifications for alerts
- Advanced analytics and reporting

### 4. What paid content, subscriptions, or features are unlocked within your app that do not use in-app purchase?

**Answer:**
None. The app does not include any paid content, subscriptions, or features that bypass Apple's In-App Purchase system. All future premium features will use Apple's In-App Purchase exclusively.

### 5. How do users obtain an account? Do users have to pay a fee to create an account?

**Answer:**
Account creation is completely FREE. Users can:
- Sign in with Apple (free)
- Sign in with Google (free)
- Use Demo Mode without creating an account (free)

No payment is required to create an account or use any features in the app.

## Free Plan Features

### Included in Free Plan (Current & Future):
- Manual website monitoring
- Monitor up to 3 sites
- Broken link detection (Quick Check & Full Scan)
- Response time tracking
- HTTP status code monitoring
- Uptime statistics
- Cloud synchronization
- Multi-device support

### Limitations in Free Plan:
- Manual monitoring only (no automatic scheduled checks)
- Limited to 3 sites
- No push notifications
- Basic analytics only

## Premium Features (Coming Soon)

### Planned Premium Features:
- **Automatic Monitoring**: Scheduled checks (hourly, daily, etc.)
- **Unlimited Sites**: Monitor as many websites as needed
- **Push Notifications**: Instant alerts for downtime or issues
- **Advanced Analytics**: Detailed reports and trends
- **Priority Support**: Faster response times

### Pricing Model (Planned):
- Subscription-based through Apple's In-App Purchase
- Exact pricing to be determined
- Free plan will always remain available
